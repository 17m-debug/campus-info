# 转专业

> 转专业的政策、流程、关键节点,以及过来人的真实经验。
> ⚠️ 转专业政策各校差异大、且常有调整,务必以本校教务处当年通知为准。

## 通用流程(参考)

| 阶段 | 事项 |
|------|------|
| 大一上 | 了解政策、明确目标专业、维持绩点 |
| 大一下 学期初 | 关注转专业通知、报名 |
| 考核期 | 笔试 / 面试 / 绩点筛选 |
| 结果公布 | 确认录取、办理学籍异动 |
| 转入后 | 补修学分、跟进新专业课程 |
