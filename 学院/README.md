# 各学院信息

本目录按学院组织,每个学院下包含四个板块:

- **就业/** —— 就业大体情况 + 个人求职经验
- **保研/** —— 保研关键节点 + 个人保研经验
- **考研/** —— 考研关键节点 + 个人考研经验
- **考试资料/** —— 本学院专业课与承担的公共课的复习资料、笔记

## 新建一个学院

复制 `数学学院/` 的目录结构,把内容替换为你学院的信息即可。每个学院应包含:

```
<学院名>/
├── README.md          # 学院总览
├── 就业/
│   ├── README.md      # 就业大体情况
│   └── 经验/
├── 保研/
│   ├── README.md      # 保研关键节点
│   └── 经验/
├── 考研/
│   ├── README.md      # 考研关键节点
│   └── 经验/
└── 考试资料/
    └── README.md      # 资料索引
```
