> 本页维护**客观信息与关键节点**;个人考研经验请见 [经验/](经验/) 目录。

## 考研关键节点(通用时间线,需按当年调整)

| 时间 | 事项 |
|------|------|
| 上一年 寒假~3月 | 确定方向、初步择校 |
| 3~6 月 | 基础阶段复习 |
| 7~8 月 | 强化阶段,确定目标院校 |
| 9 月 | 招生简章、大纲发布;预报名 |
| 10 月 | **正式报名** |
| 11 月 | 网上确认、打印准考证(临近) |
| 12 月 | **初试** |
| 次年 2 月 | 出初试成绩 |
| 次年 3~4 月 | 国家线 / 院校线公布,**复试**、调剂 |

## 数学类常见目标专业

- 基础数学 / 应用数学 / 计算数学
- 统计学 / 应用统计(专硕)
- 跨考方向:金融、计算机、数据科学等
