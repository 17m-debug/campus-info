## 保研关键节点(通用时间线,需按当年调整)

| 时间 | 事项 |
|------|------|
| 大一~大三 | 维持绩点、参与科研/竞赛、争取排名 |
| 大三寒假~3月 | 关注目标院校,准备简历、成绩单、个人陈述 |
| 3~5 月 | 各校**夏令营**报名开启 |
| 6~7 月 | 参加夏令营、争取「优秀营员」 |
| 8~9 月初 | **预推免**报名与考核 |
| 9 月下旬 | 教育部**推免服务系统**开放,正式填报、确认录取 |
| 9~10 月 | 完成接收、签三方/确认 |
