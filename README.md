# 校园信息共享 📚

> 一个由学生共同维护、面向在校生与新生的信息共享项目。
> 目标:**减小信息差**——把就业去向、保研/考研关键节点、转专业流程、考试资料和真实的个人经验沉淀下来,让后来人少走弯路。

## 这个项目能帮你什么

- 📈 了解**各学院的就业大体情况**(去向、行业、薪资区间)
- 🎓 掌握**保研 / 考研的关键时间节点**,不错过任何一步
- 🔄 看懂**转专业**的政策与流程
- 📝 找到学长学姐整理的**考试资料与经验**
- 🏫 摸清**学校生活**的各种「办事指南」

## 🤝 如何参与

这个项目靠大家一起填充才有价值。无论你是即将毕业的学长学姐,还是想分享经验的在校生:

1. 阅读 [CONTRIBUTING.md](CONTRIBUTING.md) 了解贡献方式与规范
2. 从 [templates/](templates/) 复制对应的模板
3. 填好后通过 Pull Request 或 Issue 提交

**不会用 Git 也没关系**——直接提一个 Issue,把内容贴上来,维护者会帮你整理。

## ⚠️ 重要原则

- **保护隐私**:不写真实姓名/学号,不泄露他人信息(详见 CONTRIBUTING)
- **尊重版权**:只分享个人笔记与经验,**不上传受版权保护的教材、付费课程、真题原件**
- **信息有时效**:政策与节点每年可能变化,请注意文件标注的年份,以学校官方通知为准
